## üåü Overview

**Zenith** is not just another todo app. It is a **high-performance productivity command center** designed with a "Glassmorphism" aesthetic and deep focus features. It combines the utility of Todoist with the immersive design of a gaming interface.

Built on the MERN stack (MongoDB, Express, React, Node.js), Zenith features a fully custom-built calendar system, a Pomodoro-style Focus Mode, and seamless PWA (Progressive Web App) capabilities for installation on any device.

## ‚ú® Key Features

- **üé® Zenith Ultra UI:** A stunning dark/light mode interface with "Northern Lights" animated backgrounds and frosted glass cards.
- **üìÖ Custom Date Logic:** A built-from-scratch interactive calendar popover with smart date handling (Today, Tomorrow, Next Week).
- **‚ö° Focus Mode:** An immersive, distraction-free overlay with a kinetic ring timer to help you flow through tasks.
- **üì± PWA Ready:** Installable as a native app on iOS, Android, Mac, and Windows.
- **üîç Smart Sidebar:** Quick filters, project organization, and instant task searching.
- **‚ö° Live Editing:** Inline editing for task text, dates, and priorities without leaving the list view.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

* **Node.js** (v16 or higher)
* **MongoDB** (Local installation or Atlas URI)
* **Git**

### üîß Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/iamrishit8/my-todo-app-mern.git](https://github.com/iamrishit8/my-todo-app-mern.git)
    cd my-todo-app-mern
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Configure Environment**
    Create a `.env` file in the root directory:
    ```env
    PORT=3000
    MONGO_URI=mongodb://127.0.0.1:27017/my_todo_app
    VITE_API_URL=http://localhost:3000/api
    ```

4.  **Run the App**
    This command starts both the Backend API and the Frontend Client concurrently:
    ```bash
    npm run dev
    ```

5.  **Open Zenith**
    Visit `http://localhost:5173` in your browser.

## üì± Mobile Installation (PWA)

Since Zenith is a PWA, you can install it on your phone:
1.  Navigate to your deployed URL (or local network IP).
2.  **iOS:** Tap "Share" -> "Add to Home Screen".
3.  **Android:** Tap "Three Dots" -> "Install App".
